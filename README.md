# Task-1---Data-Cleaning
Task 1- Data Cleaning (Superstore sales)
Superstore sales downloaded from kaggle
Objective
The objective of this task is to perform basic data cleaning and preprocessing on the dataset to make it ready for the analysis.

Issues Identified
Missing values in Postal code column
Date columns were in object data type
Inconsistent numeric formatting
Some irrelevant columns
Cleaning Steps performed
Checked for missing values and duplicated
Converted order date and ship date to datetime
Filled missing postal code values
Rounded sales values to two decimal places
Dropped unnecessary columns
Verified data types after cleaning
Files in this Repository
Raw data of superstore sales -> Original Dataset
cleaned data of sales -> Cleaned dataset
Tools used
Kaggle
Excel
Pandas
Google collab
Output
The final cleaned dataset is stored in 'cleaned_data.csv' and can be used for further analysis and visualisation.
